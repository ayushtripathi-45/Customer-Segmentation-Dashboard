# рҹ§  What is Customer Segmentation?

Customer Segmentation is the process of dividing customers into different groups based on common characteristics such as:

* Income
* Spending Habits
* Purchase Frequency
* Demographics

Instead of treating every customer the same, businesses can create personalized experiences for each segment.

---

# рҹӨ– What is Clustering?

Clustering is an Unsupervised Machine Learning technique used to discover hidden patterns in data.

Unlike Classification:

вқҢ No labeled output is required

вң… Algorithm automatically finds groups

Example:

Customers with similar spending behavior will naturally be placed into the same cluster.

---

# вҡҷпёҸ K-Means Clustering Explained

K-Means is one of the most widely used clustering algorithms.

### How It Works

1. Select K cluster centers
2. Assign each customer to nearest center
3. Update cluster centers
4. Repeat until convergence

### Benefits

* Fast
* Easy to understand
* Industry standard
* Highly effective for segmentation

---

# рҹ“Ҡ PCA (Principal Component Analysis)

PCA helps reduce multiple dimensions into two principal components for visualization.

Instead of visualizing:

* Income
* Age
* Spending Score
* Purchase Frequency

PCA converts them into:

* Principal Component 1
* Principal Component 2

making customer clusters easier to visualize.
